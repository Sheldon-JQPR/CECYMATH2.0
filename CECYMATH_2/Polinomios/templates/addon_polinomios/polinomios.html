¿Qué es un Polinomio?

Un polinomio es una expresión algebraica formada por términos que contienen variables elevadas a potencias enteras no negativas, multiplicadas por coeficientes.

Componentes de un Polinomio:

  • Términos: Cada parte del polinomio separada por + o -
  • Coeficientes: Los números que multiplican a las variables
  • Variables: Las letras que representan valores desconocidos (usualmente x)
  • Exponentes (Grado): Las potencias de la variable

Grado de un Polinomio:

El grado de un polinomio es el exponente mayor de la variable.

  • Grado 1: Polinomio lineal. Ejemplo: 2x + 3
  • Grado 2: Polinomio cuadrático. Ejemplo: x² + 2x + 1
  • Grado 3: Polinomio cúbico. Ejemplo: x³ + 2x² + x + 1

Ejemplo Práctico:

Evaluar el polinomio x² + 2x + 1 cuando x = 3:

3² + 2(3) + 1 = 9 + 6 + 1 = 16
