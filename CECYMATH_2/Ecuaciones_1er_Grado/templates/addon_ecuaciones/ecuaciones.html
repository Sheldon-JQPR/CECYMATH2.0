¿Qué es una Ecuación de Primer Grado?

Una ecuación de primer grado es una igualdad algebraica que contiene una variable con exponente 1. Se llama así porque la variable tiene un grado (potencia) de 1.

Forma General:

ax + b = c

  • a: Coeficiente de la variable (no puede ser cero)
  • x: La variable (incógnita)
  • b: Término independiente
  • c: El resultado de la igualdad

Pasos para Resolver:

  1. Escribe la ecuación en la forma ax + b = c
  2. Resta b de ambos lados: ax = c - b
  3. Divide entre a: x = (c - b) / a

Ejemplo Práctico:

Resolver: 2x + 3 = 11

x = (11 - 3) / 2 = 8 / 2 = 4
